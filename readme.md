# 动态星空背景项目

## 项目简介
这是一个**跨平台**的动态星空背景实现，具有以下核心功能：
- **跨平台支持**：自动适配Web、移动端和Electron环境
- **性能优化**：采用对象池和分层渲染技术，确保60fps流畅体验
- **动态场景**：根据时间自动变化的星空场景，24小时完整循环
- **天文信息**：右上角显示实时时间、星座和月相信息

## 使用说明
1. 将所有文件放在同一目录下
2. 直接用浏览器打开`index.html`
3. 无需额外依赖，纯原生JavaScript实现

## 技术亮点
- **对象池优化**：减少内存分配，提升性能
- **时间驱动场景**：根据系统时间自动过渡场景
- **响应式设计**：适配各种屏幕尺寸
- **轻量级实现**：核心代码仅约500行，易于扩展
